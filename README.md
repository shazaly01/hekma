📇 نظام إدارة البطاقات الذكي (CMS - Card Management System)نظام متكامل لإدارة وإصدار بطاقات المؤسسات الكبيرة، يجمع بين كفاءة الإدارة الورقية وسرعة التحقق الرقمي.🚀 نظرة عامةيوفر النظام حلولاً ذكية للمؤسسات التي تحتاج إلى إصدار بطاقات تعريفية (ID Cards) بكميات كبيرة، مع ميزة التحقق الفوري عبر رموز QR Code وتخصيص كامل للهوية البصرية.✨ المميزات الأساسية📦 إدارة الإصدارات: تنظيم عمليات إصدار البطاقات وتتبع حالتها (نشطة/ملغاة).🔗 تكامل QR Code: لكل بطاقة رمز فريد يوجه الزائر لصفحة ويب (Landing Page) تعرض بيانات حامل البطاقة للتحقق.🎨 تخصيص الهوية: إمكانية رفع شعارات وتصاميم مختلفة لكل بطاقة على حدة (Multi-branding support).📅 تتبع الصلاحية: سجل دقيق لتواريخ الإصدار وتنبيهات بقرب انتهاء صلاحية البطاقات.📊 لوحة تحكم إدارية: لمراقبة الإحصائيات وعدد البطاقات الصادرة لكل قسم.🛠️ البناء التقني (Tech Stack)Frontend: React.js (مع Hooks و Context API لإدارة الحالة).Backend: Node.js مع إطار عمل Express.js.Database: MySQL لتخزين البيانات العلائقية بفعالية.Auth: JWT (JSON Web Tokens) لتأمين لوحة التحكم.📂 هيكلة المشروعtext├── client/          # تطبيق React (الواجهة الأمامية)
├── server/          # تطبيق Node.js (الواجهة الخلفية)
├── database/        # مخططات SQL وملفات التهجير (Migrations)
└── README.md
